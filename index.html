    Quick Reference

    Special Characters

    • * Any value
    • , Value list (1,3,5)
    • - Range (1-5)
    • / Step (*/15)

    Field Ranges

    • Minute: 0-59
    • Hour: 0-23
    • Day: 1-31
    • Month: 1-12
    • Weekday: 0-6 (Sun=0)

    Format

    • min hour day month weekday
    • 5 fields, space separated
    • Some systems add seconds (6 fields)
